# 为人民服务的公司

对外主题：数据平权、AI 下乡。

对外品牌：码成工，一个为工友敲键盘的组织。英文名 Code for People。

软件也是一种服务，为何不还给人民。

## 我们是谁

这是一支正在筹备的小型软件团队的构想档案与第一次产品方向。

团队想在 AI 时代，为普罗大众，也就是材料里称为“工友”的人做软件，并把软件这种服务更多还给使用它、生产数据的人。

这里的“工友”不按行业划分，而按人在生产关系中的位置划分：不占有平台、数据与算法，因而不得不向其交租、出卖劳动来维持生活的一切人。包括一产二产劳动者、服务业新蓝领、个体经营者、失业待业者、无偿再生产劳动者，以及被去技能化的低级白领。

## 名字的寓意：码成工

“码成工”从“马到成功”化出：马变成码，指 code；功变成工，指工友、工人。

这个名字有两层意思：

- 以码成工：用代码成全工友，把软件这种服务还给人民。
- 码农成工：写代码的人自己也是工人。在 AI 让软件能力白菜化、码农被去技能化的今天，这是一句阶级站位的宣告。

标准署名是：码成工，一个为工友敲键盘的组织。

## 基本判断

过去二十年，互联网平台坐在生产者与消费者中间收租，活成了数字封建领主。但 AI 让写软件的能力白菜化，算力与软件产能开始过剩。

稀缺的时代，胜负看谁能生产；过剩的时代，胜负看红利让给谁。

## 四件主张

1. 数据属于用户。平台盈利向全体用户分配：看的人、创作上传的人都分，不只激励头部创作者。这就是“数据平权”。
2. AI 下乡。仿“家电下乡”。当年与今天，背景同样是过剩；把过剩的算力，用去深挖普罗大众的真实需求。
3. 普及 AI 教育，防范 AI 资本。别让 AI 重蹈土地、工业、金融、互联网资本被少数人窃取的覆辙。
4. 收入差距小于 3:1。团队内部现金收入与可支配收入，差距都不超过三倍，作为分配示范。

## 基础文件

《数据平权宣言》说明项目的观点、目的与意图。

《牛马互助协议》仿 GPL 的传染性互助协议：工友享 1/3 价；想享受同样优惠，就必须同样让利给工友。协议还附有发起方自我约束。

## 阳谋

项目不屑于隐瞒自己的观点和意图，全部打法都摆在桌面上。护城河不是秘密，而是资本结构上学不来的自我约束：资本能抄走产品，抄不走“不分红、不套现、资产归社会”。
